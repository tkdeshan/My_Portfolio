Project Details :

  • Name : Smart Home
  • About : This is the prototype of a smart home system that can securely control lights, switches, gates, and doors using a mobile phone. It utilizes sensors to detect objects and flames. The security status, sensor states, and other controlling element states can be viewed on the mobile phone.
